合作案例

为适应公司发展需要，北京基晔恒鑫动漫文化公司开始招聘，欢迎符合要求的人士投简历。

一、漫画类

线稿助理（全职3人）

1. 熟练使用（PS、SAI、CSP）等绘画软件；

2. 线条优美流畅整洁,绘画功底扎实；

3. 熟悉漫画表现语言；

4. 有良好的审美能力；

5. 有团队精神,能与人进行良好的沟通与协作。

上色助理（全职2人）

1. 擅长着色，需根据故事情节设计色彩、配色；

2. 对色彩有一定的敏感度，有良好的色彩感和色彩绘制功底；

3. 具备操作数码手写板能力，熟练使用（PS、SAI、Painter、CSP）等绘图软件；

4. 细心，有创造力，踏实上进、具备团队协作能力，善于沟通；

5. 工作有条理，能够独立并及时完成工作任务，及时交稿。

漫画主笔（全职2人）

1.绘画功底强，熟悉镜头语言，能独立完成故事漫画的分镜，画风成熟并符合大众审美，对漫画的分镜语

言有一定了解；

2.熟知场景气氛的塑造和人物表情、动态的表达，有能力把握稿件的整体画面效果；

3.善于和助理、编辑沟通，有责任心，适应团队合作；

4.能熟练使用手绘板，会熟练运用（PS、SAI、Painter、CSP）等绘画软件作画。

二、设计类

平面设计（全职1人）

1. 有较好的审美水平和平面设计能力；

2. 擅长各类风格平面设计、卡通设计、海报设计、网页设计、UI界面设计、企业VI设计等；对广告、印刷、

IT等行业都有一定的创意灵感；

3. 熟悉相关设计软件的操作（PS、AI、Coreldraw、Illustrator、InDesign等）；

4. 有一定手绘能力者优先；

5. 有相关工作经验者优先。

动画师（全职3人）

1．美术基本功扎实，强大的造型能力、人物结构和动态把握能力；

2．熟练掌握动画运动规律，良好的动画表演基础，根据前期设定及原画稿，绘制动画中间帧。根据分镜

与设计稿，为动画制作提供准确且具表演张力的原画基础能力；

3．熟练使用（FLASH、AI、PS、3DMAX、Mara ）等四、游戏类

原画设计师（全职5人）

1. 根据主美要求设计出符合游戏需求的角色、NPC、怪物、道具等造型原画设计；

2. 能协助指导3D美术人员确认造型以及把握色彩、质感等工作；

3. 擅长游戏场景设计，兼顾不同类型游戏的场景制作风格，熟悉游戏3D场景的制作流程和制作技巧，对

建筑结构、空间组合法则有比较深入的了解。能够承担PC端网游、网页游戏、移动媒体游戏中的三维场

景设计工作，也能胜任虚拟现实等行业中三维场景设计工作。

4. 具备美术基础，良好的色彩感，熟练掌握（PS、SAI、Painter、CSP）等软件。
